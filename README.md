# BookRecs

BookRecs is an app for book recommendations and reviews, where a user can add, rate, and review their favorite books. 

## Installation

```
git clone git@github.com:aut0maat10/book_recs.git
cd book_recs
rake db:migrate
rails s

```
Navigate to http://localhost:3000
